# Rock Paper Scissors - Interactive Web App

A browser-based implementation of the classic Rock Paper Scissors game, built as part of The Odin Project Foundations course.

This version extends beyond core logic into a complete interactive experience using DOM manipulation and modern UI design.

---

## Features

- Button-based gameplay (event-driven, no prompts)
- Real-time UI updates for each round
- Dynamic display of player and computer choices
- Live score tracking with first-to-five win condition
- Contextual round result messages
- SweetAlert integration for end-game feedback
- Automatic game reset after completion
- Clean, responsive UI with modern styling
- Footer with external GitHub link

---

## Tech Stack

- HTML5
- CSS3 (Flexbox, modern UI styling)
- Vanilla JavaScript (ES6+)
- SweetAlert
