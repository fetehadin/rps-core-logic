# Rock Paper Scissors — Core Logic Engine

A console-based implementation of the classic Rock Paper Scissors game, built as part of The Odin Project Foundations course.

This project focuses on mastering JavaScript fundamentals by building game logic from scratch — without relying on libraries, frameworks, or UI components.

---

## Objectives

- Practice problem-solving using a structured approach
- Strengthen understanding of JavaScript fundamentals
- Build clean, readable, and maintainable logic
- Simulate a real-world development workflow (plan → code → test)

---

## Features

- Random computer choice generation
- User input handling via prompt
- Case-insensitive input processing
- Round-based gameplay system
- Score tracking across 5 rounds
- Final winner determination

---

## Concepts Applied

- Functions and modular design
- Conditional logic (if/else)
- Scope and variable management
- String manipulation
- Debugging using the console

---

## Tech Stack

- Vanilla JavaScript
- HTML (script linkage only)

---

## Future Improvements

- Add input validation
- Refactor using loops for scalability
- Build a graphical user interface (GUI)
- Add animations and score persistence
